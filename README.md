# Komputasi Awan Project - Analisis Ulasan Gojek dengan AWS

## Deskripsi Project

Project ini bertujuan untuk membangun pipeline data berbasis cloud menggunakan AWS (S3, EC2, IAM) untuk menganalisis ulasan Gojek untuk Rating Driver. Proses dimulai dari penyimpanan data, pemrosesan di EC2, hingga visualisasi hasil analisis.

## Dataset

Dataset berasal dari Kaggle:
https://www.kaggle.com/datasets/ucupsedaya/gojek-app-reviews-bahasa-indonesia

## Struktur Repository

* `/Data` : Dataset (raw dan processed)
* `/Notebook` : Jupyter Notebook untuk analisis data di EC2
* `/Report` : Laporan akhir analisis
* `/Diagrams` : Diagram arsitektur pipeline AWS
* `/Visualisasi` : Hasil grafik dan WordCloud

## Tools yang Digunakan

* Python (Pandas, Matplotlib, Seaborn)
* Boto3 (AWS SDK)
* AWS S3 (Storage)
* AWS EC2 (Computing)
* Jupyter Notebook
* WordCloud
